# Sovereign Shards: Day One Onboarding Checklist

1. **Clone the Repository**
   - Get the project onto your computer using Git. Ask for help if you’re new to this!

2. **Run the Setup Script**
   - Run `setup.bat` to install Python, required models, and settings. No admin rights or internet should be needed.

3. **Read the Project Welcome & “Why”**
   - Read the intro message and project overview. Focus on what’s possible and why we use tiny local models.

4. **Test It Yourself**
   - Run your first tool:  
     `python tools/run/hello_world.py`  
   - You should see:  
     `Hello from ToolRegistry!`  
   - Share a screenshot or paste the output in our team chat.

5. **Claim Your First Ticket**
   - Pick an easy starter task: fix a typo, add your info to CONTRIBUTORS, or add a comment to a Python file.
   - When finished, push your changes and use the proper commit message style (help is available).

---

_Questions are encouraged! Celebrate your first commit and help others do the same._
